
# Arduino Programming Projects (C++)

This repository contains simple Arduino projects written in **C++** to demonstrate basic embedded systems programming concepts such as digital input/output, conditional logic, timing control, and interaction with hardware components.

## Project Overview

The projects focus on practical exercises using the Arduino platform, including:

* **LED Control Systems** – Turning LEDs on and off using program logic.
* **Traffic Light Simulation** – Simulating a real-world traffic light sequence using timed delays.
* **Push Button Control** – Using a button to start or stop a system.
* **LED Blinking Patterns** – Creating different blinking patterns using delays and loops.

These programs demonstrate the use of:

* Digital input and output pins
* Conditional statements (`if`, `else`)
* Loops (`for`, `while`)
* Timing control using `delay()`
* Basic hardware interaction concepts used in embedded systems

## Simulation Environment

The circuits and programs were tested using **Tinkercad Circuits**, an online electronics simulator.
This allowed the Arduino projects to be designed, tested, and debugged virtually without requiring a physical Arduino board.

## Technologies Used

* **Arduino IDE**
* **C++ (Arduino Framework)**
* **Tinkercad Circuits** for circuit simulation
* Arduino components such as LEDs, resistors, and push buttons (simulated)

## Purpose of the Repository

The purpose of this repository is to demonstrate **practical programming experience in C++ within an embedded systems environment**, focusing on simple automation and control logic.
